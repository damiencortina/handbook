# [docker ps](https://docs.docker.com/engine/reference/commandline/container_ls/)

Equivalent to :

 - `docker container ps`
 - `docker container list`
 - `docker container ls`

List running containers

**usefull options :**

| option | description |
|--|--|
| `-a` or `--all` | Show all containers |
